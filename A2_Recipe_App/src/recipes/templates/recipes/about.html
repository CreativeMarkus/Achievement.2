About This App

A recipe management application built with Django

What is My Recipe App?

My Recipe App is a full-stack web application where users can browse, create, edit, and search for recipes. It features user authentication, image uploads, search with data visualization, and a clean, responsive design. Whether you're an aspiring chef or a home cook, this app helps you organize and discover delicious recipes.

Features

Create & Manage Recipes — Add your own recipes with ingredients, instructions, cooking times, and images.

Search & Filter — Search recipes by name or cooking time and visualize results with interactive charts.

User Authentication — Secure login and user-specific recipe management.

Responsive Design — Works seamlessly on desktop, tablet, and mobile devices.

Built With

Python Django PostgreSQL Heroku WhiteNoise Gunicorn HTML/CSS Matplotlib Pandas
